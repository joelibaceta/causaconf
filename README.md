# causaconf.pe

![Node CI Status](https://github.com/peru-dev/causaconf/workflows/Node%20CI/badge.svg)

## Install

```sh
git clone git@github.com:peru-dev/causaconf.git
cd causaconf
npm i
```

## Usage

```sh
npm start
```

## Test

```sh
npm test
```

## Build

```sh
npm run build
```

## Deploy

```sh
npm run deploy
```
